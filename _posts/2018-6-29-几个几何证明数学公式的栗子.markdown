# 求解平方求和公式

1^2 + 2^2 +3^2 +……+n^2 = ？

答案 `1^2 + 2^2 +3^2 +……+n^2 = (1/6) × n × (2n+1) × (n+1)`

几何的证明是用几何体的体积证明的：

# 求解立方的求和公式

1^3 + 2^3 +3^3 + 4^3 + n^3 =？

答案 `1^3 + 2^3 +3^3 + 4^3 + n^3 = (1/4) × n^2 × (n+1)^2`
